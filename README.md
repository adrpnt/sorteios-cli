# sorteios-cli
CLI para gerar números aleatórios para os jogos Quina, Mega Sena e Loto Fácil.

### Instalação
Execute no terminal:
```
composer install
```

### Exemplo
```
./sorteios generate mega-sena
```

### Observações
O formato da resposta será sempre os números gerados separados por virgula (,).
